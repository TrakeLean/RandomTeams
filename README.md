# RandomTeams.net

A clean, modern random team generator for gaming, sports, and group activities.

## Features

- **True Random Shuffle** - Uses Fisher-Yates algorithm for fair, uniform randomization
- **Customizable Team Size** - Create teams of any size (1-10 players)
- **Incomplete Teams** - Handles odd numbers gracefully, creating teams even with fewer players
- **Responsive Design** - Optimized for desktop, tablet, and mobile
- **Modern UI** - Clean dark theme with subtle animations
- **Ad-Ready** - Built-in ad placements for monetization

## Usage

Simply enter player names separated by commas:
```
Tarek, Thomas, Minh, Alex, Jordan
```

Set your desired team size, and click "Generate Teams" to create randomized teams instantly.

## Tech Stack

- Pure HTML/CSS/JavaScript
- IBM Plex Sans font
- Fisher-Yates shuffle algorithm
- Responsive grid layout
